## Archives and Repositories (with Acquisition Strategy)

## A) Lewis County Courthouse (Weston, WV)
- Priority records:
  - Deed books (land/title chain and oil lease entries, c.1840–1905)
  - Probate records (especially estate of [[John Copley]], d. 1925)
  - Marriage registers (for Q20)
- Request strategy:
  1. Call clerk to confirm index format and copy fees.
  2. Request book/page lookup using name variants (Copley/Copely; Dolan).
  3. Order certified and non-certified scans (both if available).
- Estimated cost: $1–$3/page copies, plus certification/postage.

## B) West Virginia State Archives (Charleston, WV)
- Priority records:
  - State-level vital records and county abstracts
  - Historical newspapers for death/marriage notices
  - Land and military abstracts
- Request strategy:
  1. Search catalog first; then submit targeted pull requests.
  2. Use exact date windows derived from census/manifests.
  3. Ask staff for adjacent-reel suggestions when no-hit occurs.
- Estimated cost: travel + copy/scanning fees.

## C) National Archives and Records Administration (NARA)
- Priority records:
  - Civil War service and pension packets (Q28)
  - Quartermaster Department references (RG 92)
  - Naturalization and immigration holdings (Q19)
- Request strategy:
  1. File formal request with full identity matrix (name variants, residence, age).
  2. Request negative-search statement if no file found.
  3. Correlate findings with local county records.
- Estimated cost: depends on reproduction level; plan $30–$150+ per file set.

## D) Irish repositories (NLI + National Archives of Ireland)
- Priority records:
  - Kilgefin parish baptisms/marriages (Q1, Q3)
  - Tithe and valuation records for surname variants
- Request strategy:
  1. Build surname-variant sheet (Copley/Copely/Copple/etc.).
  2. Review by townland/parish, not by surname-only search.
  3. Capture images and full citation paths for each candidate entry.
- Estimated cost: mostly online/free; optional paid researcher if on-site needed.

## E) Diocesan/Catholic parish archives (MD/VA/WV corridor)
- Priority records:
  - Sacramental marriage entries for [[Michael Copley]] + [[Ann Elizabeth Munday Copley]] and [[John Copley]] + [[Mary Ellen Dolan Copley]].
- Request strategy:
  1. Contact diocesan archivist with narrow date windows.
  2. Include witness name possibilities and settlement clues.
  3. Ask whether records are centralized or parish-retained.
- Estimated cost: donation + search/copy fees.

## Hard-to-Find Sources: Practical Acquisition Playbook

1. **Out-of-print county books**
   - Track on AbeBooks, eBay, Biblio, and WorldCat alerts.
   - Budget: $35–$250 depending on condition/rarity.

2. **Local CD/photo archives mentioned in family documents**
   - Contact local historical societies in Lewis County.
   - Ask for inventory sheets and digitization availability.
   - Budget: variable media transfer fees.

3. **Microfilm-only records**
   - Use FamilySearch affiliate libraries and major genealogy centers.
   - Pre-order reels/collections where possible.
   - Budget: travel + printing/scanning.

4. **Newspapers behind paywalls**
   - Compare subscription cost vs. one-time lookup request.
   - Consider local library remote access credentials.
