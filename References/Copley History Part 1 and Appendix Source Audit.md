## Extraction Defaults

- Prefer **evidence/places** over broader modern-family prose enrichment.
- Prefer the **July 7, 2024** appendix file (`Part 1 Appendices .pdf`) when appendix citations are generic.
- Treat these PDFs as **family-history source material**, not automatic primary proof.
- Promote exact claims carefully:
  - figure caption / address / place description = useful source note
  - identity / migration / land-chain claims = still require confidence labeling
