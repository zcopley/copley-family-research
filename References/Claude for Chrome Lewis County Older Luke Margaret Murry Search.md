# Claude for Chrome Lewis County Older Luke Margaret Murry Search

This note preserves Zach Copley's Claude for Chrome search for the older **Luke Murry** and **Margaret Murry** named as Patrick Murry's parents in the 1871 Lewis County death register.

**Main result:** No independent digitized or indexed record was found for an older Luke Murry / Murray, born roughly 1780-1805 in Ireland, or for a wife / partner Margaret. The only FamilySearch tree entries for Luke and Margaret appear to be constructed placeholders based solely on Patrick's 1871 death record.

## Scope

Claude searched FamilySearch, WV Culture Vital Research Records, Find A Grave, and FamilySearch Family Tree for:

- Luke Murray / Murry / Murrey, born Ireland circa 1780-1815, especially in Lewis, Gilmer, or Harrison County.
- Margaret Murray / Murry in Lewis County death and collateral records.
- Patrick Murry's widow and children for later parent / grandparent clues.

Collections checked included West Virginia Deaths, U.S. census records for 1850 / 1860 / 1880, West Virginia marriages and births, WV Culture death records, FamilySearch Family Tree, and Find A Grave.

## Findings

| Target | Result |
|---|---|
| Older Luke Murry / Murray, likely born Ireland 1780-1805 | No independent record found in FamilySearch, WV Culture, Find A Grave, or indexed census / marriage / death records searched. |
| Margaret Murry / Murray, wife / partner of older Luke | No independent record found. FamilySearch tree `9HRT-TFQ` is sourced only to Patrick's death record. |
| FamilySearch tree Luke Murray `9HRT-MSZ` | Placeholder-like tree person, created from Patrick's death record only; estimated birth about 1792 is inferred, not independently sourced. |
| Younger Luke Murray `KHHZ-4BV` | Rejected as Patrick's father because the 1860 census makes him Patrick's near-contemporary. |
| Margaret Murray, died 1880, Lewis County | Rejected: parents were J. Collins and B. Collins; spouse listed as J. Murray, not Luke. |
| Ellen Murray, died 1928, Lewis County | Rejected as Patrick's daughter; parents listed Ann Lohan and Patrick White, pointing instead to the younger Luke / Anne Loghan line or a different family issue. |
| Luke Murray, died 1955, Harrison County | Patrick's son, born 17 September 1869; parents Patrick Murray and Mary Turner, likely an error for Mary Donahoe / Murray. |

## Interpretation

The older Luke / Margaret couple is still real as an informant-reported parent pair in Patrick's 1871 death register, but no independent digitized record has yet been found for them.

The search **does** separate the older Luke from the 1854 Luke. The 1860 census places Luke Murry, age 38, and adult Patrick Murry, age 34, in the same household. They are near-contemporaries, not father and son.

The most likely explanation is that Patrick's father Luke died in Ireland, died before West Virginia death registration began in 1853, or died in an unindexed / non-digitized county-record context. If the older couple left records in Lewis County, the most likely surviving paths are not broad web search but non-indexed county records: tax lists, probate / estate files, deed books, and Catholic sacramental registers.

## Next Research Target

Do not repeat the same indexed FamilySearch / WV Culture search as the next step. The next useful work is targeted non-indexed county-record work:

- Lewis County personal property tax lists before 1860.
- Lewis County land tax lists before 1860.
- Lewis County will books, estate files, and probate packets for Murray / Murry variants.
- Lewis County deed indexes / deed books after the already-completed exact grantee-index search, especially grantor-side and tax-adjacent entries.
- St. Michael's Catholic Church sacramental registers for Murray / Murry, Donahoe / Donahue, and Loghan / Logan.
