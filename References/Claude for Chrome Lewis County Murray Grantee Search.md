# Claude for Chrome Lewis County Murray Grantee Search

This note captures the April 30, 2026 Claude for Chrome handoff for the Lewis County, West Virginia compiled grantee deed index search under the exact surname **Murray**. It is a follow-up to [[References/Claude for Chrome Lewis County Deed Research Brief|Claude for Chrome Lewis County Deed Research Brief]] and should be read with [[RQ-M1-JOHN-MURRAY-DEED-FINDINGS|RQ-M1 John Murray Deed Findings]].

## Summary

Claude for Chrome searched **all 558 images** of FamilySearch DGS **8293314**, *General Index to Deeds and Oil & Gas Leases — Lewis County, W. Va. — GRANTEES* / *Deed index, grantee, G-M, 1817-1934*, for the exact surname **Murray**.

**Main result:** exact-surname Murray entries appear only on **image 553** and **image 554**, and the detailed handoff table logs **47 numbered rows**, all dated **1865-1934**. No exact-surname Murray grantee entry was reported anywhere else in the compiled index, and no exact-surname Murray grantee entry predating 1843 was found in this search.

This is an important negative result for the early-anchor question. It does **not** close RQ-M1, because:

- the search covered **exact Murray only**, not every variant
- the earlier **1826** and **1833** John Murray leads came from a different FamilySearch index path
- the Deed Book C-D built-in **"Marwee" / possible Murray** entries remain unresolved

## Caveats

- The handoff summary mentioned about 43 Murray entries from a prior pass, but the detailed final transcript above contains **47 numbered rows**. For project use, treat the detailed numbered rows as the operative record unless later image review proves otherwise.
- The FamilySearch toolbar blocked columns for image 553 rows 2-5.
- Several book/page readings remain uncertain and should be checked against deed books before citation in narrative pages.
- This handoff intentionally stops at the **index** stage. The deed bodies were **not** opened in this session.

## Interpretation

The exact-surname Murray grantee search did **not** produce an early pre-1843 anchor-family deed in the compiled grantee index. Instead, it documents a later Murray cluster from **1865 onward**, concentrated in the Cove Lick / Hannahs Camp / Sand Fork / Weston orbit and including Camden, Maxwell, Casey, Courtright, Hannigan, Doolan, and Kunst-associated transactions.

For RQ-M1, the main value of this handoff is:

1. It closes the exact-surname **compiled grantee index** pass for DGS 8293314.
2. It shows that the early 1826/1833 problem remains upstream of the later exact-Murray cluster.
3. It keeps the Deed Book C-D **"Marwee" / possible Murray** entries highly relevant.
