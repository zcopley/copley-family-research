RQ-M1 John Murray Deed Findings

This page is the findings log for the early John Murray deed leads in Lewis County, Virginia / West Virginia. It now also captures the April 28 through May 1, 2026 Claude for Chrome FamilySearch deed-index passes summarized in Claude for Chrome Lewis County Deed Research Brief, Claude for Chrome Lewis County Murray Grantee Search, Claude for Chrome Lewis County Deed Book C-D Page 404 Check, Claude for Chrome Lewis County Deed Book C-D Page 334 Check, Claude for Chrome Lewis County Built-in M Index Marwee Sawig Check, and Claude for Chrome Lewis County John Murray Fred L Fish Lease Check.

Current status: The supposed 1826 Murray / Fish lead is now closed as a FamilySearch computer-indexing error: it is actually an 1899 John Murray → Fred L. Fish oil-and-gas lease. The compiled grantee index exact-surname Murray search is complete and produced only later entries dated 1865-1934. The built-in Deed Book C-D M-index path is now retired from the Murray investigation: physical page 334 and physical page 404 are both confirmed mismatches, and the formerly live “Marwee Sewig” line now reads best as “Marwee Sawig” → John Simpson, not a Murray-variant entry. The remaining direct early deed target is now the 1833 John Murray entry. Use Tier 1 Research Execution Kit for the request and transcription template.

Research Question

Did John Murray or another Murray household hold land in the Lewis County Irish / Murray Settlement area before Michael Copley Sr. and Patrick Copley purchased land from Weeden Hoffman in 1843?

This matters because an early Murray land record near Cove Lick, Camden, Loveberry, or St. Michael’s would strengthen the hypothesis that the Murrays anchored the settlement before the Copleys arrived.

Negative Finding: Compiled Grantor Index

Claude checked the compiled Grantor Deed Index M-S, DGS 8293317, around images 55, 59, and 60. Image 60 reportedly runs from Mulvey / Mummow / Munson to Musgrave. No clear Murray entry appears in the expected alphabetic position.

This does not close the lead. It may mean the relevant records are grantee entries, were indexed under spelling variants, were missed in the compiled index, or are not the same records as the prior FamilySearch index leads.

April 30, 2026 Compiled Grantee Index Exact-Surname Results

Claude then searched the compiled Grantee Index G-M, DGS 8293314, across all 558 images for the exact surname Murray. The handoff reports that exact-surname Murray entries appear only on image 553 and image 554, and that the detailed transcript logs 47 numbered rows, all dated 1865-1934.

Key implications:

  • No exact-surname Murray grantee entry predating 1843 was found in this compiled index search.
  • The earliest reported exact-surname Murray grantee entry is 1865, including a fully readable Edward Murray / G. D. Camden entry for 174 acres on Cove Lick Fork.
  • The top-right rows on image 553 remain partly blocked by the FamilySearch toolbar, so some 1865 entries are still only partial reads.
  • This search covered exact Murray only. It does not eliminate earlier variant indexing under forms such as Murry, Murrey, Murroe, or Murrow, but the built-in Marwee path is now weakened by follow-up checks.

April 30, 2026 Physical Page 404 Check

Claude then tested the old Deed Book C-D target “Marwee Levi wife” → G.D. Camden → page 404 against the actual physical page 404 in DGS 8219256. The result is now a controlled negative finding.

Confirmed facts:

  • physical deed-book page 404 is on image 436
  • page 404 contains only wife-acknowledgment certificates for Caroline F. Green and Emma S. Fleming
  • no Camden, Murray, Marwee, or Levi appears on page 404
  • adjacent pages 396-403 also failed to show those names

Implications:

  • the old page-404 target is not a usable direct deed target as previously framed
  • the earlier image estimate of roughly 421 was wrong
  • the built-in index may still preserve a real lead, but the page reference, page-counting assumption, surname reading, or all three are unstable

See the full page-404 check note for the exact citations, URLs, and failure analysis.

April 30, 2026 Physical Page 334 Check

Claude then tested the old Deed Book C-D target “Marwee J.” → J. Peck → page 334 against the physical page assigned as 334 in DGS 8219256. This is now a second controlled negative finding in the same built-in M-index path.

Reported facts:

  • page 334 was checked on image 401, using the earlier confirmed page-404 anchor at image 436
  • page 334 contains the closing language and acknowledgment for a George Jackson deed, not a Murray-variant deed
  • no Marwee, Murray, Peck, or Camden appears on page 334 or the immediate adjacent context checked on images 400-402
  • surrounding content points instead to a North Fork of French Creek near Beaver Run context, not the Camden / Cove Lick settlement cluster

Implications:

  • physical page 334 is not a live direct deed target for the old built-in index line
  • page 334 joins page 404 as a controlled mismatch
  • the Deed Book C-D built-in M-index work now narrows to the obscured 3?? target and the underlying problem of whether the index line itself was misread or mispaged

See the full page-334 check note for the exact URLs, abstract, and failure analysis.

May 1, 2026 Built-in M Index Recheck: “Marwee Sawig”

Claude then revisited the last surviving built-in M-index line on image 10 of DGS 8219256, previously carried in this project as “Marwee Sewig” / possible Murray.

Reported facts:

  • the exact line now reads best as “Marwee Sawig” → John Simpson
  • the surname begins with a clear capital S, not M
  • the page number is still unreadable because a diagonal repair-tape strip covers the page-number column
  • exploratory checks of likely pages around 359-362 did not locate a matching deed, and page 360 was a confirmed mismatch opening with William J. Stenger

Implications:

  • this line should no longer be treated as a plausible Murray variant in RQ-M1
  • the built-in Deed Book C-D M-index path is now effectively exhausted for Murray-specific evidence
  • the remaining deed work should concentrate on the 1833 John Murray lead rather than the retired Marwee line

May 1, 2026 Murray / Fish Recheck: 1899 Lease, Not 1826 Deed

Claude then followed the long-carried FamilySearch index lead for John Murray → Fred L. Fish, previously treated here as an 1826 deed because the FamilySearch structured record showed that year.

Reported facts:

  • the index entry is in DGS 008293302, image 17, page 39
  • the deed body is fully located in the same DGS, image 52, page 39
  • the actual document is dated October 7, 1899, not 1826
  • the instrument is an oil and gas lease, not an early land-transfer deed
  • the land is about 132 acres in Lewis County, West Virginia, with neighbors including Geo. White, Thomas White, and Mrs. [Stidger?/Ridget?] Cummings

Implications:

  • the FamilySearch “1826” date is a computer-indexing error
  • this record is irrelevant to the pre-1843 Murray-anchor question
  • the old 1826 Murray / Fish slot should now be preserved as a controlled misindex / closed lead
  • the only remaining direct early John Murray deed target is now the 1833 entry

See Fish lease correction note for the exact URLs, abstract, and failure analysis.
